IPFS x402 Base

Kleros IPFS Upload

Pin to IPFS via x402 — $0.01 USDC on Base

Upload files to IPFS through the Kleros x402 payment gateway. For dispute evidence, meta-evidence, court policies, Curate item metadata, and juror justifications.

kleros-ipfs-upload/SKILL.md
Curate Ethereum Gnosis

Kleros Curate

Operate token-curated registries end-to-end

Submit items, challenge requests, fund appeals, deploy new lists. Covers Light Curate, Stake Curate (PGTCR), and Scout registries on Ethereum and Gnosis.

Cursor

Pointer · per repo

Cursor fetches when the rule fires.

1. Create this Cursor rule file 
.cursor/rules/kleros-skills.mdc
2. Paste this inside 
---
description: Kleros knowledge pack for decentralized justice
alwaysApply: true
---

Claude Code

Install · user-wide

Plugin reads from disk · granular skill loading.

1. Add the marketplace catalog 
/plugin marketplace add kleros/kleros-skills
2. Install kleros-skills (active in every project) 
/plugin install kleros-ipfs-upload@kleros-skills
