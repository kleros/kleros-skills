# kleros-skills — OpenClaw Skill

Kleros protocol knowledge for AI agents. Fetches real-time docs from [skills.kleros.io](https://skills.kleros.io) so your agent knows how to interact with Kleros dispute resolution, Curate registries, and IPFS evidence uploads.

## Install

```
clawhub install kleros-skills
```

Or copy `SKILL.md` into your OpenClaw skills directory, or point your agent at:

```
https://skills.kleros.io/SKILL.md
```

No API keys. No CLI tools. Just `curl` and read.

## Topics Covered

- IPFS evidence uploads via the Kleros x402 payment gateway
- Curate registry operations (Light Curate, Stake Curate, Scout)
- MetaEvidence retrieval and schema parsing
- Deposit computation and appeal funding
- On-chain arbitration integration
